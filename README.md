# Colad Training Examples

Training examples and inference notebooks for fine-tuned LLaMA models using Unsloth.

## Getting Started

To get started with the project:
1. Clone the repository using `git clone https://github.com/prashantbhalsingh2018-oss/Colad_Training_examples.git`
2. Open the desired Jupyter notebook in Google Colab or run Python scripts locally using your preferred environment.
